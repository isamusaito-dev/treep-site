# Treep コーポレートサイト — デザインカンプ（v2 / editorial）

要件定義書「Treep デザイン仕様 兼 構築仕様書 v1.1」とワイヤーフレームをもとに、確定ブランドカラーを当てた **静的HTMLデザインカンプ**です。後工程の **Astro + Tailwind** へそのまま移植できるトークン設計にしています。

## デザインの方向性（v2.1）
[PIECE（corp.piecehotel.com）](https://corp.piecehotel.com/) を参考にした **エディトリアル・ホスピタリティ** 路線。ブランドカラー（ティール/サンド/アンバー）は仕様v1.1のまま維持。
**行政との取引が多い**ため、信頼醸成を重視し **日本語を主体に・きっちり分かりやすく** 調整：
- **ファーストビュー（FV）に3事業の写真**を配置し、シェアヴィラ（宿泊）・農林業・イベントが一目でわかる（事業名ラベル付き）
- **ナビ・大見出し・マーキーを日本語主体**に（英語は小さなアクセントラベルに）。大見出しは日本語、その上に小さな英字ラベル
- クリーム背景のミニマル・フラット（影を多用せず余白で魅せる）
- オーガニックな4点星アクセントを画像背面に配置（ブランドモチーフ）
- アウトラインのピル型ボタン＋アンバーの塗りピルCTA、フレーム型ロゴ
- 左右非対称のエディトリアルな大判画像＋ロングテキスト、千鳥配置グリッド
- マーキーはミッション文を反復し信頼感を補強（「人と人、人と地域をつなぐ場をつくる。」等）

## プレビュー方法
```bash
cd treep-site
python3 serve.py     # http://127.0.0.1:4399/ で表示
```
または任意の静的サーバ（`npx serve` 等）でも可。各HTMLを直接ブラウザで開いても表示できます。

## 反映したブランド（仕様v1.1）
- **カラー**：ティール `#1C6E6C`（文字・見出し・基調）／ライトサンド `#F5F1E6`（背景）／アンバー `#C28F41`（CTA）。補助 SubText `#5C645E`・Border `#DED9C9`
- **施設別カラー**：うみとき `#64ACED`／やまとき `#114918`／こめとき `#D4A017`／しまとき `#1CA9A6`／おゆとき `#A3472A`／のもとき `#6A1B9A`（カードのアクセントバー・施設名に使用）
- **書体**：Noto Sans JP ＋ Montserrat（Google Fonts）
- **タイポ/余白/角丸/影**：仕様のタイプスケール・スペーシング(4pxベース)・角丸(card14 / btn8)・控えめな影を `:root` 変数化
- **ボタン**：Primary=アンバー / Secondary=ティール / Ghost=枠線。高さ48px
- **レスポンシブ**：SP〜767 / Tablet768〜1023 / PC1024〜。SPはハンバーガー＋1カラム
- **アクセシビリティ/モーション**：見出し階層・フォーカスリング・`prefers-reduced-motion` 対応

## 仮素材・要支給（差し替え前提）
- **画像はすべて Unsplash のフリー写真を仮配置**（施設・農林・イベント・ヒーロー・代表写真）。支給素材に差し替え。
- ロゴはタイポ＋簡易マークの暫定。正式ロゴデータに差し替え。
- 会社概要（所在地など）・代表経歴・実績データ・Instagram/採用リンク・フォーム通知先メールは原稿待ち（要件定義v2.0 第5章）。
